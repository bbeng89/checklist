# vue-checklist

Used on http://wheresblake.com/articles/2019/08/06/checklists/ for RV setup/teardown checklists.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
